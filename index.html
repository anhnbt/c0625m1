Hướng dẫn Git cơ bản

1. Khởi tạo repository

git init

Khởi tạo repository local (chỉ chạy 1 lần duy nhất)

2. Thêm file vào staging area

Cách 1: Thêm từng file

git add filename

Thêm từng file một (hơi lâu nhưng nên dùng)

Cách 2: Thêm tất cả file

git add .

Thêm toàn bộ file vào vùng staging cần theo dõi

3. Cấu hình tên và email

git config user.name "Tên của bạn"
git config user.email "email@example.com"

4. Tạo commit

git commit -m "Nội dung commit"

Tạo snapshot của code

5. Kết nối với remote repository

git remote add origin [URL]

Kết nối với repository trên GitHub/GitLab

6. Đẩy code lên remote

git push origin main

Đẩy code lên branch main của remote repository
